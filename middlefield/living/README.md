# Living Room – Moving Checklist

> **Room Code:** `3 – Yellow – Living Room`
> Write `3 – Living Room` on every living room box label.

## Packing Checklist
- [ ] Declutter: donate / sell / trash unwanted items
- [ ] Pack a box of essentials (snacks, clothes, toiletries)
- [ ] Disassemble furniture if necessary
- [ ] Label all boxes with contents and destination room
- [ ] Protect fragile items with bubble wrap or blankets
- [ ] Take apart electronics and label cords
- [ ] Clean the living room after packing is complete
- [ ] Do a final sweep of the room for any forgotten items
