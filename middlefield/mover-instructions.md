# Movers Instructions – 2561 Middlefield Ave

This page explains the color and number labels on all moving boxes and where they should go in the new home.

## Color & Number Legend
Use both the **number** and the **color** to quickly identify the destination room:

1. **Red – Family Room**
2. **Pink – Kitchen**
3. **Yellow – Living Room**
4. **Green – Dining Room**
5. **Orange – Pool-Facing Bedroom**
6. **Blue – Kids Room**
7. **Grey – Office**
8. **Purple – Master Bedroom**

Each box label is written as: `NUMBER – ROOM NAME` (for example: `1 – Family Room`).

## Quick Routing Script for Movers
When you see a label on a box, please use this guide:

- If label starts with **1** → Take to **Family Room** (**Red**)
- If label starts with **2** → Take to **Kitchen** (**Pink**)
- If label starts with **3** → Take to **Living Room** (**Yellow**)
- If label starts with **4** → Take to **Dining Room** (**Green**)
- If label starts with **5** → Take to **Pool-Facing Bedroom** (**Orange**)
- If label starts with **6** → Take to **Kids Room** (**Blue**)
- If label starts with **7** → Take to **Office** (**Grey**)
- If label starts with **8** → Take to **Master Bedroom** (**Purple**)

If a box label is unclear, please ask before placing it down.

Thank you for helping us keep rooms organized on move-in day!
