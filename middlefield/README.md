## Key Dates
- **Dec 2** – **Inspection day**: validate closets, garage epoxy, and garage appliances.
- **Dec 5** – Pre-moving day: you move delicate / high-value items, set up essentials.
- **Dec 6, 2:00 PM** – Official move with movers bringing furniture and remaining boxes.

> See [Dec 2 – Pre-Move Inspection Checklist](dec-2-inspection.md) for detailed checks.

### Suggested Timeline
- **Before Dec 5**
  - [ ] Confirm mover booking and arrival window.
  - [ ] Arrange elevator/parking if needed.
  - [ ] Gather packing supplies (boxes, tape, bubble wrap, labels).
  - [ ] Walk through every room and mark items for **donate / sell / trash**.
- **Dec 5 – Pre-moving (You)**
  - [ ] Move jewelry, documents, passports, checkbooks.
  - [ ] Move laptops, hard drives, and sensitive electronics.
  - [ ] Move fragile decor / sentimental items you don’t want movers handling.
  - [ ] Set up basic bedding and toiletries at 2561 Middlefield.
  - [ ] Bring a “first night” box (clothes, kids’ items, chargers, snacks).
- **Dec 6 – Movers (2:00 PM)**
  - [ ] Do a quick walkthrough with movers, room by room.
  - [ ] Guide furniture placement in each room.
  - [ ] Keep a checklist of must-open-today boxes.
  - [ ] Final sweep of old home (lights off, windows closed, no items left).
